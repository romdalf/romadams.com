# Portfolio


## Kubernetes

* ***Kleidi***: a production-grade Kubernetes KMSv2 plugin provider for HashiCorp Vault and PKCS#11 compatible HSM, superseeding the project Trousseau.io
* ***NFSv4 homes within Containers***: a research around the technical and operational challenges of implementing a secure Home directories management through NFSv4 for containerized workloads in Kubernetes environments.
* ***NetApp Consoles for OpenShift***: a set of open source native OpenShift consoles for NetApp Trident and Protect

## ML/AI
* ***ABACC-RAG***: a research around Attribute-Based Access Controlled Context for Retrieval-Augmented Generation.
* ***Dialup***: a research around FUSE client to mount ABACC-RAG content as local filesystem.
* ***Neo Deployment Strategies***: a set of deployment strategies for NetApp Neo, a data content management tool.
* ***Neo UI Framework***: an open-source UI framework supporting NetApp Neo that customer and partners can build on.
* ***Neo Fuse Framework***: an open-source Linux Fuse client to mount content from NetApp Neo as a local filesystem.
